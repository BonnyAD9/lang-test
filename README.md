# lang-test

Testing/learning languages on increasingly complex projects.

Some projects are outside of this repository because I made them as standalone
implementation independent of this initiative, but they fit here.

## Project ideas
- Prime calculator (best in [C](https://github.com/BonnyAD9/lang-test/tree/master/c/primes))
- Hexdump (best in [Rust](https://github.com/BonnyAD9/thedit))
- Brainfuck (best in [C](https://github.com/BonnyAD9/brainfuck/tree/master/c))
- Snake (best in [Odin](https://github.com/BonnyAD9/lang-test/tree/master/odin/snake))
- Fractal generator
- Plam interpreter (best in [Rust](https://github.com/BonnyAD9/plam))
- Fractal renderer (best in [C++](https://github.com/BonnyAD9/fractalio))
- Compiler
